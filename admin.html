Sign in

Paste a GitHub github_pat_. The admin runs entirely in your browser; the token is kept in localStorage and used only to read/write content.json in this repo.

To create one: github.com/settings/personal-access-tokens/new → Resource owner: viaxarmada → Repository access: select only build-a-box → Repository permissions → Contents: Read and write. Pick a short expiration (30–90 days).
